**Opdracht omschrijving**

Maak je eerste database table met PHP

Maak een table aan voor: **Begeleiders.**
Deze table bestaat uit:

1\. ID
2\. naam
3\. afdeling

**ID**
\- primary key
\- auto increment
\- nummer

**naam**
\- mag niet leeg zijn
\- string van maximaal 50 characters.

**afdeling**
\- mag wel leeg zijn
\- string van maximaal 25 characters.
